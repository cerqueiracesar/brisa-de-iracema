# brisa-de-iracema
Brisa de Iracema: Ecossistema de Inteligência de Dados para microgestão comercial. Utiliza n8n para orquestração de fluxos, IA (Gemini/LLaMA 3) para OCR de notas fiscais e processamento de linguagem natural (Telegram), integrando Google Sheets como DB e Looker Studio para visualização de KPIs financeiros e Metas de Break-even em tempo real.
